Favicon checklist

- Generate favicons (use https://realfavicongenerator.net/ for cross-platform packaged outputs).
- Place generated files into `assets/` and keep file names as referenced in `index.html`:
  - favicon-32x32.png
  - favicon-16x16.png
  - apple-touch-icon.png
  - safari-pinned-tab.svg
  - android-chrome-192x192.png
  - android-chrome-512x512.png
  - og-image.jpg (1200x630, used for Open Graph and Twitter Cards)

Tip: compress and produce WebP versions of large assets to save bandwidth. Update `index.html` if you use different file names or paths.
